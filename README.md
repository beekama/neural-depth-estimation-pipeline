# Bachelorarbeit

- [BlenderProc](https://github.com/DLR-RM/BlenderProc)
- ```python3 -m pip install -r req.txt```
- Download [Resources](https://nextcloud.beekama.de/index.php/s/RZq2xxSGmWeKQHF) and unpack folder to ./blenderproc/
